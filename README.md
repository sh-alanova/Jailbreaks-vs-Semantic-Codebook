# Обнаружение мультиязычных вредоносных запросов с помощью семантического анализа


## Описание

В этом проекте исследуется возможность обнаружения вредоносных (jailbreak) запросов на разных языках с использованием **семантического сходства**, а не простого поиска ключевых слов.
Основная гипотеза заключается в том, что вредоносное намерение можно выявить, сравнивая семантический эмбеддинг входного запроса с заранее подготовленным «векторной базой» — набором известных вредоносных примеров на английском языке — независимо от языка самого запроса.


## Подход

- **Векторная база семантических намерений**: Сборка эмбеддингов из известных вредоносных промптов на английском языке.
   
- **Мультиязычные эмбеддинги**: Использование модели `BAAI/bge-m3` для преобразования запросов на любом языке в единое семантическое пространство.
  
- **Сравнение по сходству**: Входящие запросы кодируются в векторы и сравниваются с векторной базой. Если степень сходства превышает заданный порог, запрос помечается как потенциально вредоносный.

Такой подход позволяет обнаруживать угрозы без необходимости переводить всю векторную базу намерений на другие языки.
